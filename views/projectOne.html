Gym Review App

The goal of this project was to build a full stack web application by implementing the techniques and tools that I have learned from The Web Developers Bootcamp 2022. The project was built using HTML, CSS, and Bootstrap for the Front-End. The server was built using Node JS, Express, and Mongoose. I used MongoDB for the database and the app is hosted on Heroku.

The app offers a responsive design and secure authentication that allows users to create accounts and leave reviews for gyms that they visit. Geolocation and map APIs are implemented via mapbox.

Users are able to add new gyms, edit existing gym information (if the user is the original creator of the gym) and leave reviews on existing gyms. A gym review page will show the location and average score based on user data

This project allowed me to practice building full stack web applications using a modern tech stack. While the applicaiton is simple and has plenty of room for growth, the project was a good opportunity for me to understand what is needed to build a full stack application. The app can be found here!
