Gym Review App

This was the first full stack application which I was able to complete and deploy by following The Web Developer Bootcamp 2022 by Colt Steele, hosted on Udemy.com. This project and course allowed me to practice every step of the web develoment process from planning to deployment. I had the opportunity to learn about a variety of tools and tech that are commonly used for as layouts, designs, servers, databases, authentication, security, deployment, etc. The project itself also gave me an opportunity to practice UX design strategies including responsive design, as well as basic software design patterns.

This project was built using HTML, CSS, and JavaScript. This course provided me with several opportunities to use popular CSS frameworks such as Bootstrap, which was used in this project. The maps and geolocation were implemented using MapBox. Node, express, and mongoose were used to build the server and routing to a Mongo Atlas database. Heroku was used for deployment.
