Local Business Website Redesign

This project was an assignment for my Advanced Web Design course at the University of Washington. For this project, my teamates and I were able to work with a representative from a local non-profit with the goal of redesigning a content library and curriculum for an online course that they wanted to host. We were responsible for implementing UX design strategies in an effort to restructure the clients content-heavy website into an easily managable library with structured courses and materials. We used WordPress themes and plugins, along with some HTML and CSS to build a new website for the client.

This project gave me an opportunity to design creatively while also following brand guide provided by the client. The final design included a fully organized library, instructional courses for users to follow along, updated contact and social information, authentication and authorization along with options to increase accessibility across the website.
