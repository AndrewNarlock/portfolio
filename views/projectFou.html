Wallframe Builder App

The goal of this project was to create basic blueprints for a wall frame based on dimensions provided by the user. This project allowed me to practice JavaScript and DOM manipulation while working with user input. It was a good opportunity to practice manipulating the canvas element with JavaScript functions and event listeners, which is something I plan on using for games in future projects. The app also provides an estimation for the amount of materials needed for a given blueprint, which required a bit of JavaScript calculations.

The design strategy of the project was focused around providing a lot of space for the blueprints to render clearly. The user input and navigation are meant to be minimal and responsive. The app can be seen here!
