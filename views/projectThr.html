My Portfolio Redesign

The main goal of this project was to redesign my portfolio to better represent everything I have learned from college, coding bootcamps, and self studying. My original portfolio was built using WordPress, and I wanted to use something that required more understanding of HTML, CSS, and JavaScript. The next iteration of my portfolio was built using React which I decided was not necessary for this new design. Instead, I wanted to focus more on the CSS and content layout of the portfolio, which is why I implemented Bootstrap for the CSS framework. Another massive improvement from the last website is the responsiveness of the componnets, such as the projects section carousel.

Since bootstrap carosuels are limited on displaying multiple cards at once, I had to build a custom carousel using JavaScript, CSS, and Bootstrap cards. Another cool feature is the hover effects on images located in the project pages. I was able to create responsive images that allow the user to see the images more clearly.
