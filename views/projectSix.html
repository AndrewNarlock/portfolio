Bulma CSS Framework Counter App

The goal of this project was to practice a new CSS framework called Bulma. The counter was built following a guide provided by Colt Steele in The Web Developer Bootcamp 2022 course. This project was a good opportunity to practice CSS frameworks similar to Bootstrap and helped me understand DOM manipulation. The main focus of the project was to style the counter in a way that is user friendly and responsive.
