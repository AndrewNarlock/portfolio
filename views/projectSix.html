Front-End Component Library

The goal of this project is to create resuable, front-end ui components from scratch. The components are easy to copy, paste and redesign for any small website. I plan on adding more ui components along with some back-end and routing templates in the future. The main focus is to have reusable and responsive components that use minimal code and have no dependencies.

The components are available in a github repository which can be found here! They are built so that anyone can copy and paste and edit with ease. Codepens for each component are also included in the repository.
